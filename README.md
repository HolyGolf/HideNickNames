# HideNickNames
Plugin for minecraft 1.15.2
This plugin allows you to hide and show player nicknames.

![hnames command](/images/1.png)

Commands:
  /hnames - help command
  /hnames on - hide players nick names.
  /hnames off - show players nicknames
